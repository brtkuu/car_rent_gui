# Rental Cars database management application

### Technologies

-   Node @15.7.0
-   Vue @2.5.17
-   node-firebird @0.9.9
-   Electron @2.0.2

### Application is working with Firebird database. View are generating by electron with vue.

### Run:

```
npm i
```

and:

```
npm start
```

with running firebird db in background.
